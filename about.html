What is this project about?

This project serves as a tool to brighten up your meetings by allowing you to select or create background colors. You can set the background color of your second monitor to improve lighting on your face during video calls.

Background Customization Guide

Simplify Your View with the ☀︎ Button

Easily declutter your workspace by clicking the ☀︎ Button. This feature hides the navigation bar and the main background selector, leaving you with a clean interface and a compact control box. The control box includes two essential buttons:

  • ⛅︎ – Use this button to restore the hidden elements whenever needed.
  • ❍ – Select a custom color to personalize your background. Once you've chosen a color, click "Background" to add it to the dropdown menu in the main window for future use.

How Does It Work?

  1. Choose a background color:
    • Select a pre-existing color from the dropdown menu, or
    • Use the Color Picker to create a custom shade.
  2. Click Apply to instantly update the page background.

Enjoy a more polished look and better lighting for every meeting or virtual interaction!
